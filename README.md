# AWS PHP Webアプリ（ユーザー管理システム）

転職用ポートフォリオとして作成したWebアプリケーションです。

## アプリURL
http://52.63.228.235/index.php

## 概要
AWS上に構築した、ユーザー登録・ログイン機能を持つWebアプリです。
EC2とRDSを利用し、インフラからアプリまで一貫して構築しました。

## 構成図
```
[ ブラウザ ]
     ↓ HTTP(80)
[ セキュリティグループ ]
  SSH(22) / HTTP(80) のみ許可
     ↓
[ EC2 (Amazon Linux / Nginx / PHP) ]
     ↓ MySQL(3306)
[ RDS (MySQL) ]
  別セキュリティグループで保護
  外部から直接アクセス不可
     ↓
[ VPC (172.31.0.0/20 / サブネット×3) ]
```

## 使用技術

| 分類 | 技術 |
|------|------|
| クラウド | AWS EC2 / RDS / VPC |
| Webサーバー | Nginx |
| 言語 | PHP / HTML / SQL |
| データベース | MySQL (RDS) |
| バージョン管理 | GitHub (SSH認証) |

## 主な機能
- ユーザー登録・一覧表示・編集・削除（CRUD）
- ログイン・ログアウト機能
- セッション管理
- パスワードのハッシュ化
- ログインエラーメッセージの改善

## インフラ構成の工夫
- EC2とRDSを分離し、RDSへの直接アクセスを遮断
- セキュリティグループでSSH・HTTPのみ許可（最小権限の原則）
- パスワードはハッシュ化して保存（平文保存なし）
- セッション管理によるログイン状態の制御

## 開発フロー
featureブランチを作成しPR経由でmainにマージする開発フローを実践。
コンフリクトを意図的に発生させて解消する練習も行い、約2週間でPRを8回実施。
